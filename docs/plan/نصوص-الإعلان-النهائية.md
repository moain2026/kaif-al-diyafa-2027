# 🏆 النسخة النهائية للإعلان — كيف الضيافة (دمج 4 وكلاء)

> 2026-07-21 | دمج: محلل بيانات + بحث ويب + فحص موقع/منافسين + كاتب إعلانات (كلهم Fable 5)

## 🔑 الإجماع (نقاط اتفق عليها الوكلاء = ثقة عالية)
1. **الجمع + جدة يتفوّق** على المفرد (قهوجيين 18.75% vs قهوجي 3.14%). الكلمة المفتاحية **أول العنوان** (100% جوال يقصّ).
2. **الطاقم النسائي (صبابات/مباشرات) = ذهب**: التحويل الوحيد جاء من "صبابات قهوه جده"، 52% جمهور إناث، والمنافسون لا يبرزونه.
3. **لا تثبيت (Pinning)** — يسقط قوة الإعلان (اختبار: -75% توليفات، CTR أقل).
4. **الأعراس أقوى شريحة** تاريخياً (2.05%) + العزائم.
5. **تميّز حصري (من فحص الموقع):** سقّاء زمزم + سوّاس تراثي (لا منافس يقدمها) + **خبرة منذ 2016** (أقدم من أصول الضيافة 2017).
6. **لا تنافس على "الأرخص"** (اتركها لـqahwajie) — نافس بـ"عرض سعر/استشارة مجانية" + الفخامة VIP.

## 📋 خطة التنفيذ
1. **احذف الـ15 عنوان القديمة → أدخل الـ15 الجديدة** (بدون تثبيت أي عنوان).
2. **حدّث وسائل الشرح** (استبدل بالـ10 الجديدة أو أضفها).
3. بعد الحفظ: قوة الإعلان يُفترض ترتفع لـ«ممتاز» (تنوّع + كلمات مفتاحية + بلا تثبيت).
4. لاحقاً: بنية 4 مجموعات (عام/نسائي/أعراس/عزاء) + إعلان RSA ثانٍ لكل مجموعة.
