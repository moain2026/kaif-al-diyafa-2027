# 🎯 المراجعة النقدية المدمجة (Fable 5 + Claude Sonnet 4.5)
### 2026-07-21 | الموديلان راجعا الحملة مستقلَّين — والنتيجة: اتفاق شبه تام = ثقة عالية

## ✅ إجماع الموديلين (نقاط اتفقا عليها = ثقة قصوى)

### الممتاز
- CTR 5.3% (×3.5 من التاريخ) + CPC 1.34 (تحت السقف) + 61 كلمة سلبية = بداية قوية فوق المتوسط.

### الخطر الهيكلي (كلاهما شدّد عليه)
1. **🔴 مجموعة إعلانية واحدة + إعلان RSA واحد = أخطر خلل.** خلط النوايا (قهوجيين/مباشرات/زواج/شركات) في رسالة واحدة يمنع التحسين ويآكل نقاط الجودة.
2. **🔴 إشارة التحويل غامضة:** هل «تحويل واتساب» = نقرة زر أم محادثة فعلية؟ نقرة الزر إشارة رخيصة مضلّلة. **يجب التأكد قبل أي Smart Bidding.**
3. **🔴 تحويل واحد بيوم = صفر دلالة إحصائية.** لا استنتاجات مبكرة (انتظر 50+ نقرة).
4. **🔴 «Maximize Clicks» خاطئة للهدف** (تحويلات مو نقرات) — تجلب فضوليين.
5. **🔴 كلمة «قهوجي» العامة + الكلمات المعرفية** (القهوة السعودية/البن/دلة) تهدر 30-50% من ميزانية 40/يوم.
6. **⚠️ بيانات 3.5 سنة شبه عمياء** (0 تحويل = تتبّع غائب) — يُعتمد منها فقط الديموغرافيا والتوقيت.

## ✅ الإجراءات الفورية (إجماع الموديلين، بالأولوية)
1. **أصلح إشارة التحويل** — تأكد أنها نقرة wa.me فعلية / محادثة، لا زيارة صفحة. (الأساس — إشارة فاسدة = حملة فاسدة).
2. **قسّم لـ 3-4 مجموعات إعلانية** حسب النية: (قهوجيين وصبابين) / (مباشرات وقهوجيات نسائي) / (أعراس وزواج) / (سفرجية ومناسبات شركات) — إعلانان RSA لكل مجموعة.
3. **اقتل/اكبح «قهوجي» العامة** (مطابقة عبارة، مجموعة منفصلة مراقبة) + **احذف الكلمات المعرفية** (القهوة السعودية/البن/دلة/اليوم العالمي) اليوم.
4. **جدولة العروض:** +20-30% في 6م-10م، خفض/إيقاف 12ص-9ص. تأكد الصفحة تفتح واتساب <3 ثوانٍ.
5. **RSA ثانٍ لكل مجموعة + إبراز التمايز الحصري** (سقّاء زمزم/سوّاس/طاقم نسائي/منذ 2016) — مدفون حالياً.
6. **لاحقاً (بعد 30-50 تحويل مسجّل):** انتقل لـ Target CPA / Max Conversions مع Offline Conversions (تغذية الحجوزات الفعلية).

## 💎 الرؤى الذكية (كل موديل أضاف واحدة)
- **Sonnet:** فرصة مهملة — **حفلات التخرج (مايو-يوليو) + تكريم الموظفين (ديسمبر-يناير)** — نفس الخدمة، منافسة أقل، CPC أرخص 40-60%، جمهور أسرع قراراً من العرائس.
- **Fable 5:** **المعركة الحقيقية في أول 5 دقائق داخل واتساب** — العميلة تراسل 3-4 مزودين بالتوازي؛ من يرد أولاً بعرض+صور يفوز. الحل: رد آلي فوري (باقات+صور+سؤال تأهيلي) + رد بشري <5 دقائق. ثم تغذية الحجوزات كـ Offline Conversions = الحلقة المغلقة التي تحوّل الحملة من «تصرف بلا نتيجة» إلى آلة نمو.

## الخلاصة (متفق عليها)
بداية قوية فوق المتوسط لكن **هيكلياً هشّة** (إعلان واحد + إشارة غامضة + كلمات عامة). **أصلح الإشارة والهيكل هذا الأسبوع + افتح جبهة الخرائط فوراً** — والباقي مؤجّل. لا تنتظر الكمال.
