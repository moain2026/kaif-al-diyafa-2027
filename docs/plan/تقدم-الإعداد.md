# 🔌 تقدّم إعداد Google Ads API — كيف الضيافة (موقوف مؤقتاً، نكمّله لاحقاً)

## ✅ ما تم إنجازه
1. **حساب مدير (MCC):** جاهز. رقم الحساب المدير (MCC ID): **يُلتقط من الحساب** (ظهر 794-756-6294 مبدئياً — تأكيد لاحق).
2. **Developer Token:** ✅ تم الحصول عليه.
   - ⚠️ **التوكن ظهر في لقطة شاشة — يجب إعادة ضبطه (Reset) لاحقاً لأنه انكشف.**
   - المستوى الحالي: **Test Access** (اختباري — لا يعمل على الحساب الحقيقي بعد).
3. **ملف توثيق الأداة (PDF):** جاهز في `API-TOOL-DESIGN.pdf` (لسؤال 8 في نموذج Basic Access).

## ⏳ المتبقّي (نكمّله لاحقاً)
1. **Google Cloud project:** إنشاؤه من `console.cloud.google.com/projectcreate` (اسم مقترح: keif-aldiafa-ads) → أخذ **رقم المشروع** (12 خانة).
2. **نموذج Basic Access:** إكماله وإرساله (يحتاج رقم Cloud + ملف PDF). الإجابات الجاهزة:
   - سؤال 3 (MCC ID): رقم حساب المدير.
   - سؤال 4: keifaldiafa@gmail.com
   - سؤال 5: No | سؤال 6: https://keifaldiafa.com
   - سؤال 7: (الوصف الإنجليزي في API-TOOL-DESIGN.md القسم 2).
   - سؤال 9: Internal users only | سؤال 10: No | سؤال 11: No.
   - مراجعة قوقل: ~5 أيام عمل.
3. **OAuth credentials:** من Google Cloud (Client ID + Secret + Refresh Token).
4. **السكربت:** Python مع google-ads library (يسحب تقارير + يدير كلمات/سلبيات/مزايدة).

## 🔗 روابط مهمة
- مركز API: https://ads.google.com/aw/apicenter
- إنشاء Cloud project: https://console.cloud.google.com/projectcreate
- Google Ads Scripts (بديل أسهل بدون API): https://ads.google.com/aw/bulk/scripts?ocid=1167578585
- سكربت جاهز محفوظ: `google-ads-script.js` (يضيف الكلمات السلبية تلقائياً)

## ملاحظة
البديل الأسرع للتحكم الآلي دون API = **Google Ads Scripts** (JavaScript داخل الحساب، بلا توكن/Cloud). السكربت جاهز في `google-ads-script.js`.
