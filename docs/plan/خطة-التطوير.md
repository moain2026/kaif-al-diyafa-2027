# 📈 خطة العمل المتكاملة للتطوير — كيف الضيافة
### (خطة العميل/عمر — موثّقة 2026-07-21)

## أولاً — تطوير الموقع وبناء صفحات متخصصة (الأكبر والأهم)
بناء صفحات مخصصة تستهدف كل خدمة في كل منطقة تغطيها — كل صفحة بمحتوى فريد وصور خاصة وكلمات مدروسة، لالتقاط أكبر عدد من عمليات البحث وتصدّر النتائج. ليس نسخ ولصق — كل صفحة مشروع بحد ذاته (هنا القوة والاحترافية). يأخذ وقتاً متوسطاً ليطلع باحترافية تنافسية.
- **ربط ببياناتنا:** البحث يثبت نمط «قهوجي + حي» (الفيحاء/المحمدية/الزهراء). صفحات الأحياء تلتقطه مباشرة. الموقع أصلاً مبني على 40 صفحة SEO (خدمة×مدينة) — نبني عليها ونوسّع للأحياء.

## ثانياً — تعميق محتوى الموقع
تطوير وتعميق المحتوى باستمرار لتصدّر محركات البحث والظهور في إجابات الذكاء الاصطناعي (GEO/AEO) — شغل تحريري وتقني بمتابعة دائمة.

## ثالثاً — السوشال ميديا والمحتوى
إنتاج محتوى احترافي من الأعمال (مقاطع قصيرة تيك توك/سناب/انستقرام) — قطاع الضيافة بصري، والانتشار البصري يقوّي الاسم.
- **ربط ببياناتنا:** الجمهور 25-44 (80%) + إناث 52% + 100% جوال → المقاطع القصيرة تصل لهم مباشرة.

## رابعاً — الحملات الإعلانية
إدارة حملات مدروسة + تتبّع كل عميل يتواصل لمعرفة ما ينجح وتطويره.
- **الحالة:** ✅ منجز جزئياً — حملة Search مطلقة ومحسّنة (يوليو 2026). التتبّع (واتساب AW-11081441847) يعمل.

## خامساً — تعزيز الظهور في خرائط جوجل (مهم جداً محلياً)
تفعيل وتطوير ملف النشاط التجاري (Google Business Profile) للظهور لمن يبحث محلياً، مع نظام يجمع تقييمات العملاء — مهم جداً للظهور المحلي.
- **ربط ببياناتنا:** 🔴 الأهم — البحث محلي 100% (أحياء جدة) + التحويل نقرة واتساب. ملف الخرائط + التقييمات يظهرك مجاناً فوق النتائج المحلية = مضاعفة العملاء بأقل تكلفة.

## 📊 تسلسل الأولوية المقترح (رأي مهني)
1. **خرائط جوجل + التقييمات** (أسرع أثر، أقل تكلفة، بياناتنا تؤكده) — ابدأ به فوراً.
2. **الحملات الإعلانية** (منجز — نستمر في التحسين).
3. **صفحات الأحياء المتخصصة** (أثر تراكمي قوي، يأخذ وقت).
4. **تعميق المحتوى + AI** (مستمر مع الصفحات).
5. **السوشال ميديا** (بناء علامة طويل المدى).
